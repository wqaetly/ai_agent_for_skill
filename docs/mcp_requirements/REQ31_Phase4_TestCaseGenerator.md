# REQ-31 Phase4 自动生成测试用例

## 背景
- 需要自动生成技能测试用例以验证配置和回归。

## 目标
- 根据技能结构生成测试场景（输入动作、期望帧事件、验证断言）。

## 实施步骤
1. 解析技能JSON，提取关键步骤与依赖。
2. 按Action类型映射到测试模板（Animation、Damage、Control等）。
3. 生成Unity可执行的测试脚本或伪代码，并附期望结果。
4. 与CI集成，可一键运行关键技能测试。

## 验收标准
- 至少覆盖MVP技能，生成用例可在编辑器中执行。

## 风险
- Unity自动化环境复杂 → 先输出脚本模板，再逐步接入自动运行。

## 交付
- 用例生成器、模板库、CI说明。
