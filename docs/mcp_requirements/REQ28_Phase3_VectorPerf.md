# REQ-28 Phase3 向量检索性能优化

## 背景
- 需要提升技能检索性能以支撑更多实时请求。

## 目标
- 降低search_skills_semantic延迟，支持批量查询与更高QPS。

## 实施步骤
1. 调整Chroma索引参数（HNSW M/ef）并评估延迟-召回权衡。
2. 引入分片或并行查询策略，结合缓存。
3. 打通批量接口，允许一次请求多个query。
4. 建立性能监控，观察P50/P95延迟与资源占用。

## 验收标准
- P95延迟降低30%，QPS提升2倍。

## 风险
- 索引调整影响召回率 → 构建回归测试确保质量。

## 交付
- 优化配置、性能测试报告、监控Dashboard。
