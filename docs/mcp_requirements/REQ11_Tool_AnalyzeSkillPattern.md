# REQ-11 MCP Tool: analyze_skill_pattern

**状态**: [P2-可选] | **优先级**: 低 | **备注**: MVP 后根据需求决定是否实施

## 背景
- 需要识别技能模式（Combo/Burst等）来驱动推荐与优化建议。

## 功能目标
- 输入skill_id，输出pattern_type、特征描述、相似技能、复杂度分析。

## 实现步骤
1. 提取技能时间轴和Action序列，计算节奏/协同等特征。
2. 基于规则或分类模型映射到预定义模式。
3. 查找相似技能并计算复杂度指标（timing_precision等）。

## 数据与依赖
- 使用索引器统计、Action分类器以及RAG引用示例。

## 验收标准
- 典型技能识别准确率≥90%。
- 输出至少3条模式特征描述。

## 风险
- 模式定义不足 → 引入可配置词典并支持人工标注扩展。

## 交付
- 模式库、分析脚本、示例报告、验证集。
