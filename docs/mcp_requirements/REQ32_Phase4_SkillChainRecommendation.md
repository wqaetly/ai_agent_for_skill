# REQ-32 Phase4 技能链组合推荐

## 背景
- 高级功能需推荐技能链（多个技能/Action组合），辅助设计技能连招。

## 目标
- 根据技能标签/模式推荐可串联的技能链，并评估协同效果。

## 实施步骤
1. 构建技能关系图（标签、效果、冷却）。
2. 定义组合规则（前置/后置、冷却兼容、资源占用）。
3. 搜索符合条件的技能链，计算协同评分并给出理由。
4. 输出JSON/文本描述，供AI助手讲解。

## 验收标准
- 每次调用至少返回1条链路，并附置信度+解释。

## 风险
- 组合空间爆炸 → 限制链长度并加入剪枝策略。

## 交付
- 关系图数据、推荐算法、示例报告。
