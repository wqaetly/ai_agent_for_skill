# REQ-25 Phase2 Inspector实时推荐

## 背景
- 在Phase2中需实现Inspector侧实时展示AI推荐，提升交互效率。

## 目标
- 当选中Action/轨道时自动刷新推荐/推理结果，并支持手动刷新。
- 支持结果高亮、差异比对、置信度提示。

## 实施步骤
1. Unity插件监听Selection变化，收集上下文并调用MCP Tool3/4。
2. 设计UI组件显示推荐列表、参数表与依赖警告。
3. 支持一键应用/撤销，与Tool6返回的验证信息同步显示。
4. 记录用户采纳/拒绝行为，反馈给后端用于评估。

## 验收标准
- 切换Action后1s内呈现最新推荐。
- 应用失败时清晰提示原因并保留原值。

## 风险
- 大量请求导致卡顿 → 使用节流+合并请求策略。

## 交付
- Inspector扩展面板、交互文档、可用性测试记录。
