# REQ-26 Phase2 可视化参数调试工具

## 背景
- 为帮助设计师理解参数影响，需要提供可视化调试工具（Timeline/图表）。

## 目标
- 展示Action时间轴、参数曲线、冲突提示，与AI推荐联动。

## 实施步骤
1. 从技能JSON构建Timeline模型，标注帧/Action类型。
2. 为关键参数（伤害、范围、移动等）绘制曲线或柱状图。
3. 将Tool6警告与Tool4推荐叠加在可视化上，支持跳转到原始配置。
4. 支持快照对比（当前 vs 推荐）。

## 验收标准
- 可视化刷新耗时 < 200ms。
- 用户可从图表跳转至具体Action并一键应用建议。

## 风险
- 数据量大导致渲染卡顿 → 限制展示窗口并做虚拟化渲染。

## 交付
- 可视化面板、渲染组件、交互说明。
