# REQ-08 MCP Tool: infer_action_parameters

## 背景
- 是最核心的参数推理工具，需要结合统计、规则和上下文生成可落地的参数集。

## 功能目标
- 输入action_type与skill_context，返回参数值、置信度、理由、替代值、引用技能、依赖提示。

## 实现步骤
1. 拉取相似技能并提取对应Action参数样本集。
2. 计算统计量（中位数、分位点），应用领域规则调整。
3. 产出ParameterRecommendation对象，包含值、区间、解释、参考技能。
4. 输出parameter_dependencies作为下游验证提示。

## 接口示例
见计划原文示例。

## 数据与依赖
- 依赖REQ02上下文特征、RAG搜索、统计缓存。

## 验收标准
- 结果中≥80%参数置信度>0.75。
- 提案必须包含至少1条具体引用技能或规则原因。

## 风险
- 统计样本偏斜 → 引入异常值过滤与manual override。

## 交付
- Tool实现、统计模块、规则库、回归测试和监控仪表。
